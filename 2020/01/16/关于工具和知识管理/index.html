关于工具和知识管理

发表于 2020-01-16 更新于 2020-01-19 分类于 开发者 阅读次数： Valine：
本文字数： 1.4k 阅读时长 ≈ 1 分钟

如果你也是计算机专业的话，相信对于一些开发工具都有了解

而且学习了相关知识，自然也需要把一些重要的东西记录下来

总结一年来学习的各种知识点，年初目标完成了多少，明年又要计划做什么？期间可能要翻翻自己的笔记、代码

这里说一下我使用的吧！

开发工具

python

看过侧边栏的关于的内容的都知道，我也学 python 的吧 虽然也是新手

如果说搞这个有什么好工具的话，嗯~

最开始了解的应该是 IDLE

这个是下载 python 后自带的一个东西，初期使用还可以，界面相当相当的简洁 跟个记事本一样 ，不过后期使用这个不推荐吧，看的难受啊不觉得吗

最好使的应该就是 pycharm

这个不陌生吧？ 这是使用率最高的编写 python 的软件，相当方便，内置功能也很齐全

它分为两个版本，简单的说就是免费版和收钱版。

社区版(免费)就可以满足一般的需求了，如果觉得还不够就加钱吧

前端

我自己使用的是 HBuilderX 这款软件，个人认为吧，这个用来写这些很方便

一些你能想到的功能它基本都支持，而且界面还蛮好看的

如果需要什么插件的话，也可以在内部一键安装，相当方便

而且它还可以用来开发 app 和小程序

Visual Studio code

一个运行于 Mac OS X、Windows和 Linux 之上的，针对于编写现代 Web 和云应用的跨平台源代码编辑器

用起来也不错，这个轻量化编辑器

也可以用来写写 markdown 真的是多功能爽歪歪

需要什么插件也可以快速的在搜索里找到并安装

知识管理

脑子不可能把你学习过的东西全给记得一清二楚吧？有的时候你也会忘记一些很重要的东西

不是有句话这么说的吗，好记性不如烂笔头

如果你看过我博客的评论区域的话，会发现它是支持 markdown 的，其实我记录这些资料也是使用 markdown 的

对于使用 markdown 语法来记录，推荐的软件是有道云，它支持新建 markdown 文档，而且还支持同步功能，不仅可以随时随地查看，还可以把笔记分享给别人。

兼容大部分 Markdown 平台（简书、Github、Hexo）的渲染样式

Gitgub

一个平台，你可以把自己的代码托管在上面保存，而且你还可以通过这个平台参加一些大佬的项目，来提高自己的能力

并且你还发掘一些有趣的代码并把它们拉取到自己仓库，通过 git cloen 指令克隆到本地查看

它还可以发布静态页面，你的一些网页就可以放到上面发布，我的博客就是托管在 Github 上面，还支持绑定域名

不过因为访问速度比较慢，可以使用 CDN 来进行加速

码云

这个好像之前有个水文讲过来着，其实这个和上面的 Github 在功能上差别并不大，但是码云的访问速度很快

这个尤其体现在使用 git 进行克隆的时候，比使用 Github 仓库克隆的时候快得多

我拿博客上的图片，就是存放在码云的

而且码云可以同步 GitHub 仓库，解决了后顾之忧

它同样也可以发布静态页面，不过免费版有很多限制了

pro 版的价格是 99元/年

coding

这个是一个类似团队合作的平台个人觉得，如果你的团队5个人以下就是免费，但是超过5个人就是要收费的，按团队规模来收费

相对来说还是可以，而且它同样可以部署静态网页

只不过这东西老是容易抽风，有时候静态网页莫名其妙的打不开了WQ

而且现在部署的话，它还一个代码审核，这个是真的无语，有事没事就说你的哪个里面非法，气都会气死在这里

要不是Github国内访问速度慢的离谱我会用你？
