v2ray搭建小飞机

发表于 2020-04-02 分类于 v2ray 阅读次数： Valine：
本文字数： 1.6k 阅读时长 ≈ 1 分钟

V2Ray是近几年兴起的科学上网技术，采用新的协议，因功能强大、能有效抵抗墙的干扰而广受好评

相对于SSR更加稳定，下面就来说说怎么使用v2ray来搭建小飞机吧

服务器准备

首先你需要一台部署在中国大陆之外的服务器，你要是用部署在国内的服务器那是没什么用的

vultr是一个好选择，但是最近好像查的严，经常会出现IP被墙的情况，它的好处是它使用的是按时长计费的方式

万一被墙了就换一台，反正它是按时长计费的

这里推荐使用阿里云的轻量应用服务器，比vultr要稳定的多，不会出现连接半天连不上服务器的问题

进入阿里云的轻量应用服务器的购买界面，地域选择中国香港或者新加坡(推荐香港)，系统镜像选centos，套餐看你自己的需求而选择了

选择完成后，支付就行，之后等待产品开通

服务器内准备

等开通成功后，进入阿里云的控制台，远程连接你的服务器

连接成功后就可以开始搭建了，这一步很简单，直接使用一键搭建脚本就行

复制下面的命令到服务器并运行

bash <(curl -Ls https://blog.sprov.xyz/v2-ui.sh)

若出现 bash curl: command not found 等错误，请先安装 curl
CentOS 系统：yum install curl -y
Debian 或 Ubuntu 系统：apt install curl -y

更新与升级都是这条命令，升级不会造成数据丢失，你所做的个性化面板设置也会保留。

全新安装后，在浏览器中打开 http://服务器IP:65432 即可访问面板，默认用户名和密码都是 admin

当然，之前最好看看你服务器的安全组有没有放行端口，不然会出现访问不了的情况

升级后，仍然使用你升级前的方式进入面板。

面板操作

进入面板后，就可以看见运行情况了

点击账号列表，再点击左上角的 +

只需要把其中的传输协议改成 ws 就行，其他的不要动，备注随意填写

然后点完成，别忘了在服务器安全组里把对应的端口放行

到此服务器这边就完成了，下面配置客户端

客户端下载直接点目录里的客户端，然后按你自己的平台下载就好

客户端配置

PC

因为我是 win 平台所以就说说 win 上面怎么配置

打开 v2rayN，点击服务器，创建 vmess 服务器

地址(IP)，端口，ID，额外id都按照你账号列表里，创建的账号的参数来配置

加密方式选 auto，传输协议选 ws

然后保存，接下来对着任务栏的应用图标按右键，Http代理里选择 开启PAC，并自动配置PAC(PAC模式)

如果你浏览器里启用了类似谷歌上网助手的插件，最好将其关闭

当 v2rayN 下面出现类似 tcp:127.0.0.1:63167 accepted tcp:mtalk.google.com:5248 [proxy] 之类的字样，就说明成功了

移动端

客户端的下载依旧和上面的一样，顺带说一下 ios 里相关的软件已经下架了，如果想使用的话得自己找找了

手机上配置相当简单，在面板的账号列表里，点击账号旁边的二维码图标，就会出现一个二维码，然后软件里直接扫描二维码

就可以完成自动配置了，然后启用就行

是不是相当简单

ws + tls

这个方法可以提高你的飞机速度，不过前提条件是你要有个域名

获得域名的方法这里就不提及了。直接说配置方法

首先需要给你的域名申请SSL证书，这个百度一下就可以找到很多，阿里腾讯也提供相关服务，你有它们的域名可以直接申请

申请完成后，打开面板，在账号列表里点开你的账号配置，打开 tls ，域名这里填写你申请证书的域名

下面证书这里点击证书文件内容，证书文件内容这里把你的SSL证书内容复制进去，下面的密钥就把你的私钥内容复制进去(-----BEGIN PRIVATE KEY----- 开头的)

完成后保存即可

当然你客户端也是需要改配置的，客户端中的地址改为你的域名，伪装类型 none，路径里填 /，底层传输安全选 tls，allowinsecure让它保持为空

保存就行，至于手机端就再扫一次二维码就行了
